class linkml_renderer.renderers.html_renderer.HTMLRenderer(style_engine=None)

Bases: Renderer

A renderer that generates HTML.

To generate HTML for a YAMLRoot object, use the render() method:

>>> from linkml_renderer.renderers.html_renderer import HTMLRenderer
>>> from linkml_runtime import SchemaView
>>> import yaml
>>> sv = SchemaView('my-schema.yaml')
>>> renderer = HTMLRenderer()
>>> with open('my-instance.yaml') as f:
>>>     instance = yaml.load(f)
>>>     print(renderer.render(instance, sv))

This method can be configured using a Configuration object

>>> from linkml_renderer.style.style_engine import StyleEngine
>>> se = StyleEngine(sv, configuration = my_config)
>>> renderer.style_engine = se

render(element, schemaview, source_element_name=None, **kwargs)

Dump a YAMLRoot object to HTML.

The input may be a YAMLRoot object, a pydantic BaseModel, or a dict. It must conform to the LinkML schema provided via the schemaview argument.

Passing in a source_element_name is optional if this can be inferred from the schema.

Parameters

  • element (Union[YAMLRoot, BaseModel, Dict[str, Any]]) – instance to render

  • schemaview (SchemaView) – describes the structure of the instance to render

  • source_element_name (Optional[str]) – name of the element type the instance instantiates.
